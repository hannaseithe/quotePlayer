Managing your Quotes

Here you can create, import, edit and delete your Quotes.

warningIn order to "play" your quotes in the QuotePlayer Popup, you will first have to add them to a playlist, which you then will select in the QuotePlayer Popup.

To add a new Quote click on the Link in the blue navigation bar, fill out the input fields (only the "Quote" field is obligatory) and click on the sendsend Button. The new quote will be automatically added.

To import Quote(s) from Excel click on the Link in the blue navigation bar, select the excel file that has been prepared by you on your computer, and then click on the orange button . The Quotes will be automatically added.

In the Quotes Table you can view all the Quotes that currently exist in your Database.

You can edit edit or deletedelete each Quote

You can filter Quotes. Just enter the word by which you want to filter in the corresponding input field. E.g. If you want to show all Quotes by Nietzsche, you enter "Nietzsche" in the "Filter Authors" input field.

You can sort the quotes, by clicking on the arrow_upward/arrow_downward arrow that shows when you hover over the Filter input fields.
