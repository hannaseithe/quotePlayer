How to import Quotes from an Excel File

Click on the folder Icon and then choose the excel file you want to import from your computer.

2. Click on the enabled "Import Quotes from Excel" Button (next to the folder icon)


The file can contain only one table and that table has to follow certain specifications (otherwise the "Quote Player" wont be able to interpret the data correctly).

the first row (1) has to contain four fields (from A to D) with the EXACT words:
quote | author | source | tags

the quote column always has to be filled out. The other columns can be left empty

if you want to add more than one tag for one quote, you have to seperate the tags by ", " - e.g. "responsibility, vision"
